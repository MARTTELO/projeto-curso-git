    o git não se importa com os arquivos que não foram comitados e não vai monitorar

    ===========================================================================

  • git reset - remove os arquivos que foram adicionado da standinh area do git
  • git add -A - adiciona tudo que foi modificado para a stading area
  • git commit -a -m 'texto' - comitando sem fazer o add
  • git add . - adiciona tudo da pasta que está pronta para ser adicionada, o que estiver fora da pasta não será adicionado
  • git commit -C 'HASH DO COMMIT' -m 'texto' - comitando alterações com o mesmo commit da última alteração.
  • git commit --amend --no-edit -> comita as alterações modificando o commit anterior, sem alterar o texto do commit
  • git log - Mostra o historico de commits
  • git log --oneline - mostra os comits na forma reduida, 1 commit por linha
  • git reset 'nome-do arquivo-alterao'- remove o comite d standig area
  • git restore . - deleta totas as alterações feitas
  • git revert - deleta o comit e as alterações ficam disponiveis para salvar na stading novamente
  • git revert HEAD~2 reverte antepenultimo comit
  • git pull - puxa o repositorio do github
  • git push - envia o repositorio local para o github
  • git remote add 'origin' linkgithub -> origin é sua maquina
  • git branch -M main - altera o nome da branch principal para main
  • git push -u origin main - enviar seus arquivos para o repositorio no github
  • git branch 'nome-da-branch' - criar branch
  • git branch -l -> listar as branchs
  • git branch --show-current -> exibir branch atual
  • git checkout 'nome-da-branch' -> trocar de branch atual para a nomeada
  • git checkout -b 'nome-da-branch'-> criar uma nova branch e muda pra ela
  • git switch 'nome-da-branch' -> trocar de branch
  • git branch -M 'nome-da-branch' -> renomear branch
  • git branch --merge - mostra o merge a partir do ramo atual
  • git branch --no-merge -> mostra os branchs que não firzeram merge com a ramo atual
  • git branch -D 'nome-da-branch' - deletar a branch; Vc precisa estar numa branch diferente da que quer deletar
  • git push origin --delete 'nome-da-branch' -> vai deletar a branch do repositorio

      …or create a new repository on the command line

      ===========================================================================

    1. echo "# projeto-curso-git" >> README.md
    2. git init
    3. git add README.md
    4. git commit -m "first commit"
    5. git remote add origin 'link repositorio github'
    6. git push -u origin master

      ===========================================================================

      …or push an existing repository from the command line

      ===========================================================================

    1. git remote add origin 'link repositorio github' -> estabelece conecao com o github
    2. git branch -M main -> renomeia a branch principal para main
    3. git push -u origin main => envia os arquivos para o repositorio remoto
